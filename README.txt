

This repository contains code to perform no-linear square fittings to retrieve viscoelastic information from static force spectroscopy AFM data.
The details can be found in:
López-Guerra, Enrique A., Babak Eslami, and Santiago D. Solares. "Calculation of standard viscoelastic responses with multiple retardation 
times through analysis of static force spectroscopy AFM data." Journal of Polymer Science Part B: Polymer Physics 55.10 (2017): 804-813.
